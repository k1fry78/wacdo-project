Données personnelles

Conformément à la réglementation en vigueur, vous disposez d'un droit de consultation, de modification et de suppression de vos données personnelles.

Vous pouvez exercer ces droits en nous contactant via les moyens suivants :

  • Par email : support@wacdo.com
  • Par téléphone : +33 1 23 45 67 89
  • Ou directement depuis votre espace utilisateur dans les paramètres de votre compte.

Nous nous engageons à traiter votre demande dans les meilleurs délais.

Pourquoi collectons-nous vos données ?

Nous collectons vos données personnelles pour les raisons suivantes :

  • Pour améliorer nos services et personnaliser votre expérience utilisateur.
  • Pour vous informer des mises à jour importantes concernant votre compte ou nos services.
  • Pour respecter nos obligations légales et réglementaires.

Comment protégeons-nous vos données ?

La sécurité de vos données est notre priorité. Voici les mesures que nous prenons pour les protéger :

  • Utilisation de protocoles de sécurité avancés pour le stockage et le transfert des données.
  • Accès limité aux données uniquement aux employés autorisés.
  • Audits réguliers pour garantir la conformité avec les normes de sécurité.

Vos droits

Vous avez le droit de :

  • Accéder à vos données personnelles.
  • Demander la correction de données inexactes ou incomplètes.
  • Demander la suppression de vos données personnelles.
  • Limiter ou vous opposer au traitement de vos données.

Pour exercer ces droits, veuillez nous contacter via les moyens mentionnés ci-dessus.
